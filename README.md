# longest-path-search
最長片道きっぷの旅/ドリーム・アーツ コーディングテスト

## 概要
無向グラフにおいて、同じ点を2回通らない最長経路を探索する。
始点と終点が同じになる閉じた経路も探索範囲に含める。

## 実行方法
```
javac PathSearcher.java
java PathSearcher < input.txt
```

## 入力フォーマット
`始点ID, 終点ID, 距離`の形式で1行ずつ標準入力から与える。

## 探索の方針
全頂点を始点として再帰的に経路を探索する。
進める隣の頂点へ進み続け、行き止まりに来たら戻って別の道を試す形で全経路を網羅し、その中で最長のものを記録する。
